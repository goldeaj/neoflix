# StreamFlix - Streaming Website

A modern, Netflix-inspired streaming website with a video hero section and web series image cards.

## Features

- **Video Hero Section**: Full-screen video background with overlay content featuring a highlighted series
- **Web Series Cards**: Multiple rows of series cards with hover effects and interactive buttons
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Interactive Elements**: 
  - Carousel navigation for browsing series
  - Mute/unmute button for hero video
  - Play, add to list, and like buttons on each card
  - Smooth scrolling and animations
  - Navbar that changes on scroll

## How to Use

1. Open `index.html` in your web browser
2. Browse through different series by hovering over cards
3. Use the arrow buttons to navigate through carousels
4. Click play buttons to simulate video playback
5. Toggle video sound with the mute/unmute button in the hero section

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - JavaScript for interactivity and animations

## Customization

### Adding Your Own Content

1. **Replace placeholder images**: Update the `src` attributes in the `<img>` tags with your own series posters
2. **Change hero video**: Replace the video source in the hero section with your own video file
3. **Update series information**: Modify the titles, descriptions, and metadata in the HTML

### Color Scheme

The website uses CSS variables that can be easily customized in `styles.css`:
- `--primary-color`: #e50914 (red accent color)
- `--bg-dark`: #141414 (dark background)
- `--bg-darker`: #0a0a0a (darker background for footer)
- `--text-primary`: #ffffff (white text)
- `--text-secondary`: #b3b3b3 (gray text)
